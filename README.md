# 🚀 AstroVerse – Space Exploration Portal

AstroVerse is an educational HTML-based website dedicated to space exploration. It provides information about planets, rockets, space missions, astronauts, and other fascinating topics related to the universe.

## 🌌 Features

* 🪐 Explore the Solar System and planets
* 🚀 Learn about famous rockets and launch vehicles
* 🛰️ Discover historic and modern space missions
* 👨‍🚀 Read about legendary astronauts and space pioneers
* 🌠 User-friendly navigation with multiple linked pages
* 📖 Educational content suitable for students and space enthusiasts

## 📂 Project Structure

## 🛠️ Technologies Used

* HTML5
* Semantic HTML
* Images and hyperlinks

> This project is intentionally built using only HTML to demonstrate a strong understanding of webpage structure and semantic markup.

## 🚀 Getting Started

1. Clone the repository:

```bash
git clone https://github.com/your-username/AstroVerse.git
```

2. Open the project folder.

3. Open `index.html` in your preferred web browser.

No additional software or installation is required.
